## HistAi

Цель проекта — разработка ИИ-чат-бота, который имитирует диалог с исторической личностью, опираясь на реальные исторические документы, письма, указы, мемуары и научные источники.

Базовой персоной для проекта выбран Пётр I, так как:

- он хорошо изучен историками;
- существует большой объём оцифрованных первоисточников;
- его язык, стиль и взгляды достаточно ярко выражены.

## Ожидаемый результат:

- веб-интерфейс и Telegram-бот для общения с ИИ-персонажем;
- диалоговая система, поддерживающая исторически корректные ответы в рамках эпохи;
- возможность сравнения различных LLM и подходов (prompt engineering, RAG);
- прототип, который может использоваться
  - в образовательных целях (школы, вузы, онлайн-курсы)
  - для популяризации истории
  - как демонстрация применения RAG и LLM в гуманитарных задачах.

## Примерный график работы по проекту

Февраль – начало марта

- исследование источников по Петру I. Цель набрать побольше настоящих данных, глубоко погружаться в них на этом этапе не будем: нейронки любят много данных, и имеет смысл как можно раньше собрать большой датасет, дабы не терять время на это на более поздних этапах
- выбор базовых языковых моделей
- проектирование архитектуры

Март

- реализация базового чат-бота для тестирования во время разработки
- первичные промпты для few-shot реализации: тут уже надо качественно разобрать полученные данные
- сравнение выбранных моделей
- Возможно(!!!) прототип RAG (векторное хранилище, retrieval). (Мы не уверены, что данных для применения RAG окажется достаточно, так что если их получится собрать достаточно много, то попробуем реализовать RAG, иначе - останемся на базовом пути, который, как мы ожидаем, и так окажется более эффективным)

Апрель

- улучшение качества диалога засчет модификации промптов
- разработка веб-интерфейса и окончательной версии Telegram-бота
- добавление дополнительных личностей для диалогов(Если основная версия будет работать достаточно хорошо и при этом будет время)
- тестирование и отладка

Май

- продолжение тестирования и отладки (да, это может занять много времени :) )
- подготовка к защите и полировка
- завершение проекта до конца мая
